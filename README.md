## üß≠ Overview

**Leafly** is an interactive web application designed for indoor plant enthusiasts.
It combines a **plant marketplace**, **educational resources**, and **personalized user profiles** in one seamless platform.

Users can:

* Browse curated collections of indoor plants
* Access detailed care instructions and expert tips
* Book consultations with plant care specialists
* Manage personal profiles, including photo and display name
* Authenticate securely using email/password or Google login

The platform emphasizes **smooth navigation**, **real-time updates**, and **mobile-first responsive design** using modern technologies.

## üß© Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/Riad-Zz/Leafly
cd Leafly
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run Development Server

```bash
npm run dev
```

### 4. Environment Variables

Create a `.env` file in the root directory:

```
VITE_apiKey=
VITE_authDomain=
VITE_projectId=
VITE_storageBucket=
VITE_messagingSenderId=
VITE_appId=
```
